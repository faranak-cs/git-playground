# Git playground

Basic for Git

## Git commands

- Check status
  `git status`

- Check all branches
  `git branch -a`

- Check current branch
  `git branch`

- Create new branch
  `git branch branch_name`

- Switch required branch
  `git checkout branch_name`

- Stage all the changes
  `git add .`

- Stage specific file
  `git add file_name`
